LearnMate AI is an AI-powered study assistant chatbot designed to help students understand concepts related to Artificial Intelligence and Machine Learning. The chatbot provides interactive responses through a simple web interface and demonstrates the integration of LLM-based logic using modern web technologies.

## Features

1. AI-based study assistant chatbot
2. Backend built using FastAPI
3. Frontend built using Streamlit
4. REST API-based communication
5. Easy to extend with real LLM APIs
6. Beginner-friendly project structure

##  Tech Stack

1. Programming Language: Python
2. Backend: FastAPI
3. Frontend: Streamlit
4. LLM Logic: Rule-based (can be replaced with LLM API)
5. Version Control: Git & GitHub

## System Architecture

1. User enters a query in the Streamlit UI
2. Frontend sends request to FastAPI backend
3. Backend processes the input and generates response
4. Response is returned and displayed in UI
